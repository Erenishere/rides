# FC Rides - Assets Folder

This folder contains all the images and media files used in the FC Rides website.

## Folder Structure

```
assets/
├── imgs/
│   └── fleet/           # Fleet vehicle images
│       ├── sedan-1.jpg  # First Class Sedan
│       ├── sedan-2.jpg  # Premium Sedan
│       ├── suv-1.jpg    # Luxury SUV
│       └── van-1.jpg    # Executive Van
└── README.md           # This file
```

## Image Specifications

### Fleet Images
- **Dimensions**: 280px × 175px (16:10 aspect ratio)
- **Format**: JPG or PNG
- **Quality**: High resolution for crisp display
- **Naming Convention**: [vehicle-type]-[number].jpg

### Fallback System
All fleet images include placeholder fallbacks that will display if the local images are not found:
- Uses via.placeholder.com service
- Matches the exact dimensions (280x175)
- Uses brand colors (#2596be background, white text)
